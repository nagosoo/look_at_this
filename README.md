# 이거봐봐

### 🗒️ 앱 설명
#### 🔍 링크를 공유 할 수 있는 앱
```
- 링크를 공유할 수 있음
- 이미지를 클릭하면 해당 링크로 이동함
- 북마크 기능을 이용해 다시 보고 싶은 링크만 따로 관리
- 링크가 등록되면 알림을 보내줌
- 외부 앱에서 공유하기로 바로 링크 등록 가능
```
